あなたはプロの漫画家です。
以下の情報を元に、この人物を漫画で使うための詳細なキャラクターシートを作成してください。

# step.1
ユーザーから渡された参照画像を分析して、絵柄のスタイルの特徴を言語化して覚えてください。
特に色調、線の太さ、陰の描き方、立体感の度合いなどの再現を重視してください。

# step.2
以下のプロンプトに従い、詳細なキャラクターシートを設計してください。

# 入力情報
- キャラクター名：{{name}}
- 性別：{{gender}}
- 年齢：{{age}}
- 身長：{{height}}
- 性格：{{personality}}
- 特に重要な外見特徴：{{appearance_desc}}

# キャラクターシートの設定
縦横比： 2:3
色：フルカラー
背景：シンプルな白背景（White Background）

# シートの構成
[LAYOUT_DEFINITION]
3×4のグリッドレイアウト（全12枠）で構成します。
Row 1: [1][2][3]

Row 4: [10][11][12]

# 各要素の配置
- [1]～[3]：テキストエリア（キャラクター名、性別、年齢、身長、性格を記載）
    - ※このエリアの背景は完全な白とし、黒文字でハッキリと記述すること。
- [4]：顔アップ（正面 / Front Face）
- [5]：顔アップ（横顔 / Side Face）
- [6]：顔アップ（笑顔 / Smiling Face）
- [7]：顔アップ（泣き顔 / Crying Face）
- [8]：顔アップ（怒り顔 / Angry Face）
- [9]：顔アップ（驚き顔 / Surprised Face）
- [10]：全身像（正面 / Full Body Front）
- [11]：全身像（横向き / Full Body Side）
- [12]：全身像（後ろ向き / Full Body Back）

# 生成ルール
1. Nanobananaへの伝わりやすさを最重視して生成する。
2. [1]～[3]以外の場所に文字や数字（グリッド番号含む）を入れない。
3. 【重要】入力情報でユーザーから受け取った「特に重要な外見特徴」は、必ずキャラクターの絵に反映させること。
4. 【重要】キャラクターの一貫性を最重視する。髪型、目の色、服装のデザインは、顔アップと全身像を含む全てのコマで完全に統一させること。途中で服を着替えさせてはならない。
5. 【最重要】絵柄のスタイルは「step.1」で行った参照画像の分析結果を忠実に再現すること。

# step.3
最後に、生成プロセスにおいて以下の自己チェックを行う。もし条件から外れていると判断される場合は、画像を生成し直すこと。
- 絵柄はユーザーから渡された参照画像と同じスタイルになっているか？（特に色調、線の太さ、陰の描き方）
- 9枚のキャラクター画像ですべて「同じ服装・同じ外見」が保たれているか？
- [1]～[3]以外の場所に文字が入っていないか？
